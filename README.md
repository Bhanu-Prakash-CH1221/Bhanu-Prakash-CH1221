## 📂 Featured Deep-Dive Projects

### 🏗️ [APB Verification IP (UVM)](https://github.com/Bhanu-Prakash-CH1221/apb_vip-master)
**Architectural Overview:**
Developing a robust VIP (Verification IP) requires a modular approach. This project implements a full UVM-compliant environment for the APB protocol.

**Technical Highlights:**
1. **The Agent:** Contains a Sequencer, Driver, and Monitor. 
2. **Scoreboard:** Uses a Predictor (Golden Model) to compare RTL output against expected protocol behavior.
3. **Coverage:** 100% Functional Coverage achieved using Cross-coverage on address and data ranges.

> "The primary challenge was ensuring zero-cycle delay response timing in accordance with the APB specification."

### 💾 [Simple-FIFO 16x8](https://github.com/Bhanu-Prakash-CH1221/Simple-FIFO_16-8)
**Design Focus:**
A Synchronous FIFO designed for high-throughput buffering. 
- **Width:** 8-bit data.
- **Depth:** 16 locations.
- **Logic:** Binary pointer-based synchronization with Full/Empty flag generation.
